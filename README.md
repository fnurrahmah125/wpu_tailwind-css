# Tailwind CSS

A Utility-first CSS Framework that packed with classes

## Utilities

- Preflight
- Layout
- Flexbox & Grid
- Spacing & Sizing
- Typography
- Background & Borders
- Effect & Filters
- Transition, Transformation & Animation
- Tables
- ...

## Features

- Interactivity
  - Pseudo-classes
    - hover, focus, active, first, last, ...
  - Pseudo-element
    - before, after, placeholder, file, selection, ...
- Responsive design
- Dark mode
- Reusability
- Custom Styles
- Functions & Directives
